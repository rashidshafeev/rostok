# **Документация по использованию `Smartbox`**

`Smartbox` — это JavaScript-класс для создания интерактивных всплывающих элементов (попапов), которые можно настраивать по положению, содержимому, стилям и поведению. Этот компонент поддерживает автоматическое позиционирование, закрытие по клику вне, а также динамическое обновление содержимого.

---

## **Создание экземпляра**

### **Конструктор**

```javascript
new Smartbox(smartboxSelector, options);
```

- **`smartboxSelector`** (обязательный): уникальный идентификатор, указанный в атрибуте `data-smartbox` элемента.  
  *Пример*: `<div data-smartbox="example"></div>`

- **`options`** (необязательный): объект настроек, который задает поведение и внешний вид `Smartbox`.  

### **Опции (`options`)**

| Опция              | Тип       | По умолчанию  | Описание                                                                                 |
|--------------------|-----------|---------------|-----------------------------------------------------------------------------------------|
| `side`             | `string`  | `'bottom'`    | Сторона, где появится контент: `'top'`, `'bottom'`, `'left'`, `'right'`.                |
| `align`            | `string`  | `'start'`     | Выравнивание: `'start'`, `'center'`, `'end'`.                                           |
| `closeOnScroll`    | `boolean` | `false`       | Автоматически закрывать попап при скролле страницы.                                     |
| `content`          | `string`  | `''`          | HTML-содержимое, отображаемое в попапе.                                                |
| `contentClass`     | `string`  | `''`          | CSS-классы, применяемые к содержимому.                                                 |
| `parent`           | `string`  | `smartboxName`| Атрибут `data-smartbox-parent`, определяющий контейнер, где будет размещаться контент.  |

---

## **Пример использования**

1. **HTML-разметка:**

```html
<div data-smartbox-parent="example">
   <div data-smartbox="example">Нажми меня</div>
</div>
```

2. **Инициализация Smartbox:**

```javascript
const smartbox = new Smartbox('example', {
  side: 'top',
  align: 'center',
  content: '<p>Это контент Smartbox</p>',
  contentClass: 'custom-class',
});
```

3. **Обновление содержимого:**

```javascript
smartbox.setContent('<p>Новое содержимое</p>');
```

4. **Добавление/удаление классов к контенту:**

```javascript
smartbox.addContentClass('highlight');
smartbox.removeContentClass('highlight');
```

---

## **Методы**

### 1. **`showContent()`**

Показывает содержимое Smartbox.  

```javascript
smartbox.showContent();
```

### 2. **`closeContent()`**

Закрывает содержимое Smartbox.  

```javascript
smartbox.closeContent();
```

### 3. **`setContent(htmlContent)`**

Обновляет HTML-содержимое всплывающего окна.  

- **`htmlContent`**: строка с новым содержимым.

```javascript
smartbox.setContent('<p>Новое содержимое</p>');
```

### 4. **`addContentClass(className)`**

Добавляет CSS-классы к всплывающему содержимому.  

- **`className`**: строка с именем класса.  

```javascript
smartbox.addContentClass('highlight');
```

### 5. **`removeContentClass(className)`**

Удаляет указанные CSS-классы из всплывающего содержимого.  

- **`className`**: строка с именем класса.  

```javascript
smartbox.removeContentClass('highlight');
```

---

## **Описание работы**

1. **Инициализация:**
   - После создания экземпляра `Smartbox`, он привязывает обработчик клика к элементу, указанному в `data-smartbox`.  
   - При клике отображается содержимое, заданное в опции `content`.

2. **Закрытие:**
   - Всплывающее содержимое автоматически закрывается при клике за его пределы или вызове метода `closeContent`.

3. **Позиционирование:**
   - Содержимое автоматически располагается относительно элемента в зависимости от опций `side` и `align`.

4. **Обновление содержимого:**
   - Содержимое можно динамически обновлять с помощью метода `setContent`.

5. **Стилизация:**
   - Используйте опцию `contentClass` для добавления пользовательских стилей.

---

## **Рекомендации**

- **CSS-стили:** Убедитесь, что вы стилизовали `.custom-class` или другой переданный класс. Например:

```css
.custom-class {
  background: white;
  border: 1px solid #ddd;
  padding: 10px;
  border-radius: 4px;
}
```

- **Динамическое обновление:** Если содержимое зависит от взаимодействий, используйте метод `setContent`.

- **Контейнер для контента:** Если не указывать `parent`, содержимое будет размещено в `body`.
